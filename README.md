# Customer-Churn-Analysis-and-Retention-Strategy
 

1. Project Overview
This project analyzes customer churn behavior using exploratory data analysis (EDA).
The goal is to identify key factors influencing churn and provide actionable business
recommendations to improve customer retention.

2. Dataset
- Source: Telecom Customer Churn Dataset
- Records: ~7,000 customers
- Features include contract type, payment method, tenure, internet service, and churn status.

3. Key Insights
- Month-to-month customers show the highest churn rate (~42%)
- Electronic check users churn more (~45%)
- Customers with less than 1 year tenure have ~50% churn
- Senior citizens churn more than non-senior customers

4.Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

5. Visualizations
- Churn by Contract Type
- Churn by Payment Method
- Churn vs Tenure
- Percentage-based stacked bar charts

Conclusion
Long-term contracts and automatic payment methods significantly reduce churn.
Early customer engagement is critical for retention.
